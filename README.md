# Staff Work Skills

> Military-grade thinking tools for Claude Code. Cut through complexity, communicate clearly, act decisively.

A curated collection of [Claude Code](https://claude.ai/code) skills that apply military analytical frameworks to everyday work — planning, writing, and decision-making.

---

## Skills

### Analysis

| Skill | What it does | Say... |
|-------|-------------|--------|
| [main-effort](analysis/main-effort/SKILL.md) | Strips competing priorities down to one main effort and supporting tasks | "what's the main effort", "help me prioritise", "everything feels urgent" |
| [pocket-estimate](analysis/pocket-estimate/SKILL.md) | Rapid military analysis — 7 Questions crushed to 3, SMEAC brief in under a minute | "quick estimate", "help me think this through", "what should I do next" |
| [task-purpose](analysis/task-purpose/SKILL.md) | Converts any tasking into Task–Purpose format: *Do X in order to Y* | "task this out", "write a directive", "what should I tell them to do" |

### Style

| Skill | What it does | Say... |
|-------|-------------|--------|
| [active-voice](style/active-voice/SKILL.md) | Converts passive voice and nominalisations into direct, active prose | "make this more direct", "active voice this", "who does what?" |
| [bluf-writer](style/bluf-writer/SKILL.md) | Rewrites any draft as a military-style BLUF memo — bottom line first | "BLUF this", "make this bottom line up front", "turn this into a memo" |

---

## Installation

Clone the repo and copy any `SKILL.md` into your Claude Code skills directory, or reference it from your project's `.claude/` config:

```bash
git clone https://github.com/harry-burton-dev/staff-work
```

---

## Usage

Once installed, Claude Code will automatically detect and apply skills when you use the trigger phrases listed above. You can also invoke them explicitly:

```
/main-effort
/pocket-estimate
/task-purpose
/active-voice
/bluf-writer
```
